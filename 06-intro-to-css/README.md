# Intro to CSS

CSS (Cascading Style Sheets) is a declarative language that controls how webpages look in the browser.


## Including CSS

1. Inline Styles
2. Internal Styles (`<style>` Element)
3. External Stylesheet


## CSS Rules

```css
selector {
  property: value;
}

h1 {
  color: purple;
}

img {
  width: 100px;
  height: 200px;
}
```


## Some Element Selectors

- `body`
- `h1`
- `h2`
- `ul`
- `a`
- `footer`
- `p`


## Basic Properties

- `color`
- `background-color`
